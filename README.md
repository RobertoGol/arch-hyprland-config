## Arch + Hyprland (VMware) — Автоматическая установка

Этот репозиторий содержит **единый скрипт полностью автоматической установки** Arch Linux с Hyprland и VMware Tools в виртуальной машине.

Скрипт:
- **стирает и разметит диск `/dev/sda`** (EFI + SWAP + ROOT);
- устанавливает **Arch Linux, ядро, GRUB**;
- настраивает **локаль, время, hostname, пользователя и sudo**;
- ставит **Hyprland + базовые приложения + Firefox Developer Edition + VMware Tools**;
- при наличии папки `config/` в репозитории **копирует конфиги в `~/.config`** нового пользователя;
- **автоматически настраивает оптимизации производительности** (sysctl, I/O scheduler, лимиты системы);
- **оптимизирует pacman** для ускорения установки (параллельные загрузки, цветной вывод).

### ВАЖНО

- Скрипт предназначен для **VMware** (UEFI) и **полностью очищает `/dev/sda`**.  
  Не используйте его на диске с важными данными / dualboot.
- Запускать только в **Live-среде Arch**, загруженной с официального ISO.

## Быстрый старт (одна команда)

В Live-среде Arch выполните **одну команду**:

```bash
pacman -Sy git --noconfirm && \
git clone https://github.com/RobertoGol/arch-hyprland-config.git && \
cd arch-hyprland-config && \
chmod +x all_in_one_install.sh && \
./all_in_one_install.sh
```

Дальше всё делается **полностью автоматически**:
- скрипт автоматически создаст пользователя Archlinux/Archlinux и root/root;
- автоматически разметит диск, установит систему, Hyprland, Firefox Developer Edition и конфиги;
- настроит оптимизации производительности системы;
- всё произойдёт без вашего участия.

Подробнее пошагово то же самое описано в `install_guide.txt`.

## Структура репозитория

- **`all_in_one_install.sh`** — главный скрипт полной автоматической установки.
- **`install_guide.txt`** — короткое текстовое руководство по установке из Live-среды.
- **`USER_AGREEMENT.txt`** — пользовательское соглашение (EULA).
- **`config/`** — конфиги для автоматической настройки окружения:
  - `hypr/` — конфигурация Hyprland (оптимизирована для производительности)
  - `waybar/` — конфигурация и стили Waybar
  - `kitty/` — конфигурация терминала Kitty
  - `wofi/` — конфигурация и стили Wofi (лаунчер)
  
  Всё содержимое этой папки будет скопировано в `~/.config` нового пользователя.

## После установки

После завершения скрипта:
- настройте в VMware **Boot Order**, чтобы диск ВМ был выше ISO;
- загрузитесь с установленного диска;
- войдите под пользователем:

  - обычный пользователь: **Archlinux / Archlinux**
  - root: **root / root** (используйте только при необходимости, лучше работать от пользователя + sudo)

- запустите Hyprland командой:

```bash
Hyprland
```
