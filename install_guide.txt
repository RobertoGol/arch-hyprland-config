# =========================================================
# ARCH LINUX + HYPRLAND: КРАТКОЕ РУКОВОДСТВО ДЛЯ VMware
# =========================================================

# ВАЖНО:
# - Скрипт all_in_one_install.sh автоматически ОЧИСТИТ и разметит /dev/sda.
# - Используйте его только на тестовой/чистой ВМ, где диск можно полностью стереть.

# --- ШАГ 1: ПОДГОТОВКА СРЕДЫ (РУЧНАЯ КОМАНДА) ---

# В Live-среде Arch (после загрузки с официального ISO) выполните ТОЛЬКО установку git:

pacman -Sy git --noconfirm

# --- ШАГ 2: КЛОНИРОВАНИЕ РЕПОЗИТОРИЯ И ЗАПУСК СКРИПТА ---

# 1. Клонируйте репозиторий и перейдите в каталог:
git clone https://github.com/RobertoGol/arch-hyprland-config.git
cd arch-hyprland-config

# 2. Сделайте скрипт исполняемым и запустите:
chmod +x all_in_one_install.sh
./all_in_one_install.sh

# Скрипт полностью автоматический:
# - создаст пользователя Archlinux с паролем Archlinux;
# - установит пароль root как root;
# - автоматически установит hostname: arch-hypr-rj;
# - автоматически разметит и отформатирует /dev/sda (EFI + SWAP + ROOT);
# - установит Arch Linux, Hyprland, Firefox Developer Edition, базовые приложения и VMware Tools;
# - скопирует конфиги из папки config/ (если она есть в репозитории) в ~/.config;
# - настроит оптимизации производительности системы.

# --- ШАГ 3: ПОСЛЕ ПЕРЕЗАГРУЗКИ ---

# 1. ПЕРЕД перезагрузкой: ОБЯЗАТЕЛЬНО настройте в VMware "Boot Order",
#    чтобы жесткий диск ВМ был первым, а ISO-образ - вторым или ниже.
#    Это гарантирует, что система загрузится в установленный Arch Linux.
# 2. Войдите в систему под созданным пользователем.
# 3. Запустите Hyprland командой:
# Hyprland

# Если конфиги не были скопированы автоматически:
# - перейдите в каталог репозитория (например, ~/arch-hyprland-config);
# - вручную скопируйте содержимое папки config в ~/.config.
